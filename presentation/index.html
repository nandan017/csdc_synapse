Under the hood
Built with intention.
⚛️

Next.js 14

App Router, server components, middleware auth, API proxy routes.

⚡

FastAPI

Python backend. Async routes. Clean REST API. Handles all business logic.

🗄️

Supabase

PostgreSQL + Auth + Storage. Row-level security. Real-time capable.

📡

Web NFC API

Browser-native NFC reading. AES-encrypted UIDs. Zero app installs.

🚀

Vercel

Frontend deployed on Vercel. Edge middleware for auth. Auto-scaling.

📧

Brevo

Transactional emails for invite links. Templated, reliable delivery.
